ctrlp-tagbar.vim
====================

Extension for ctrlp.vim to allow integration between tagbar and ctrlp

No need to configure ctags in two places if it is already tuned well in tagbar.


Requirement
-----------

- [ctrlp.vim](https://github.com/ctrlpvim/ctrlp.vim)
